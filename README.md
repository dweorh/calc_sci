# calc_sci

It is a simple scientific calculator written in C++.  
It calculates equations in RPN (Reverse Polish Notation).
Eg.

equation: 2+2\*2  
RPN: 2 2 2 \* +  
result: 6  

2sin(90)+5!  
RPN: 2 90 sin * 5 ! +  
result: 122  

arcsin(arccos(arctan(tan(cos(sin(9))))))  
RPN: 9 sin cos tan arctan arccos arcsin  
result: 0.027419
